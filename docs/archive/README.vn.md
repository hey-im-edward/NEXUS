## 💡 Tại sao lại là NEXUS?

**Bạn có bao giờ cảm thấy mình đang phải thích nghi với công cụ thay vì công cụ phù hợp với mình?**

Hầu hết các ứng dụng năng suất ép bạn vào quy trình làm việc của _họ_:

- 📝 **Todoist** tuyệt vời cho task đơn giản — nhưng nếu bạn cần CRM thì sao?
- 📊 **Notion** rất linh hoạt — nhưng xây dựng workflow phức tạp mất hàng giờ
- ⚙️ **ClickUp** có đầy đủ tính năng — nhưng tốn $10+/người và mất nhiều ngày để học

**NEXUS khác biệt.**

Thay vì phải chọn giữa đơn-giản-nhưng-hạn-chế hoặc mạnh-mẽ-nhưng-phức-tạp, bạn có:

✨ **Xây dựng ứng dụng riêng** với các component kéo-thả (không cần code)
🎨 **Sắp xếp workspace** chính xác như bạn muốn (giống màn hình chính iPhone)
🏪 **Chia sẻ & khám phá** ứng dụng do cộng đồng xây dựng (như App Store)

### Cách hoạt động

```
1. Chọn Layout              2. Xây dựng hoặc Cài đặt App    3. Tùy chỉnh mọi thứ
┌─────────────────┐         ┌─────────────────┐             ┌─────────────────┐
│  [App] [App]    │         │  • Kéo Button   │             │  Đổi màu sắc    │
│  [App] [App]    │    →    │  • Thêm Text    │     →       │  Sửa layout     │
│  [App] [App]    │         │  • Kết nối DB   │             │  Workflow của   │
└─────────────────┘         └─────────────────┘             │  riêng bạn      │
                                                            └─────────────────┘
```

## 🎯 Bạn có thể xây dựng gì

### Ứng dụng có sẵn (Sử dụng ngay)

- ✅ **Task Manager** - Theo dõi công việc với priority, due date, và kanban board
- ✅ **Quick Notes** - Ghi chú nhanh ngay lập tức (tự động lưu vào browser)
- ✅ **Pomodoro Timer** - Tập trung với chu kỳ làm việc 25 phút
- ✅ **Today's Tasks** - Xem công việc hết hạn hôm nay từ tất cả dự án

### Ứng dụng tùy chỉnh (Tự xây dựng)

Với **App Builder No-Code**, bạn có thể tạo:

- 📊 **Expense Tracker** - Quản lý ngân sách với danh mục riêng của bạn
- 👥 **Simple CRM** - Theo dõi khách hàng không cần độ phức tạp của Salesforce
- 📅 **Content Calendar** - Lên kế hoạch đăng bài cho Instagram, Twitter, LinkedIn
- 🎯 **Habit Tracker** - Xây dựng chuỗi thói quen và trực quan hóa tiến độ
- 💡 **Idea Board** - Ghi lại và sắp xếp ý tưởng sáng tạo
- 📈 **Dashboard** - Kết hợp các widget vào workspace hoàn hảo của bạn

### Sắp ra mắt

- 🏪 **Community Marketplace** - Cài đặt ứng dụng do người khác xây dựng (như App Store mini)
- 🧩 **Low-Code Mode** - Thêm logic điều kiện ("nếu task khẩn cấp, gửi thông báo")
- 🔗 **Database Integration** - Kết nối với nguồn dữ liệu riêng của bạn
- 👥 **Team Sharing** - Cộng tác với đồng đội trên workspace chung

## 🛠 Tech Stack

Xây dựng bằng công nghệ hiện đại, đã được chứng minh:

- **Frontend:** Next.js 16 + React 19 + TypeScript
- **Backend:** Supabase (PostgreSQL + Auth + Realtime)
- **UI:** Tailwind CSS + shadcn/ui components
- **Deployment:** Vercel (deploy tức thì, CDN toàn cầu)

**Tại sao chọn những công nghệ này?**

- ⚡ **Phát triển nhanh** - Tech stack thân thiện với AI, tài liệu tuyệt vời
- 💰 **Hosting $0** cho MVP (free tier đủ cho mọi thứ)
- 🔒 **Đã được kiểm chứng** - Được sử dụng bởi các công ty như Vercel, Supabase, Airbnb
- 📚 **Tài liệu tốt** - Dễ học, dễ đóng góp

## 🚀 Bắt đầu nhanh

### Yêu cầu

- **Node.js** 18+ và npm/pnpm/yarn
- **Tài khoản Supabase** ([đăng ký miễn phí](https://supabase.com))
- **Git**

### Cài đặt

```bash
# Clone repository
git clone https://github.com/hey-im-edward/NEXUS.git
cd NEXUS/frontend

# Cài đặt dependencies
npm install

# Thiết lập biến môi trường
cp .env.example .env.local
# Sửa .env.local với thông tin Supabase của bạn

# Chạy database migrations (từ thư mục supabase)
cd ../supabase
supabase db push

# Khởi động development server
cd ../frontend
npm run dev
```

Mở [http://localhost:3000](http://localhost:3000) trong trình duyệt.

### Biến môi trường

Tạo file `.env.local` trong thư mục `frontend`:

```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

Lấy thông tin từ [Supabase Dashboard](https://app.supabase.com/) → Project Settings → API.
